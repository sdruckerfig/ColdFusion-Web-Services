CFJQM Setup Instructions:

1) Install ColdFusion 10
2) Copy CFJQM folder into wwwroot
3) Install MySQL and MySQL WorkBench
4) Use MySQL Workbench to restore cfjqmdb.mysql.backup 
5) Create ColdFusion datasource named "cfjqm" that points to the restored db
6) Open Google Chrome to http://localhost:8500/cfjqm/walksolution/final
  (The SubGenius University Application should load)

ENJOY!
